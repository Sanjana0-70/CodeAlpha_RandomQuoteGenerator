# Random Quote Generator

A polished, interactive quote app built to inspire and motivate your day. Get instant access to 30 carefully curated quotes from world-renowned leaders and thinkers, with a beautiful interface and effortless sharing.

## ✨ What This App Does

This app lets you:

- discover 30 inspiring quotes from influential figures
- reveal new quotes instantly with one click
- copy any quote to your clipboard with a single tap
- share quotes easily with friends and colleagues
- enjoy smooth animations and a beautiful modern interface
- get motivated anytime, anywhere in your browser

## 🚀 Features

- Random quote generation from curated collection
- One-click copy to clipboard with visual feedback
- Smooth animations and transitions
- Fully responsive design (desktop, tablet, mobile)
- Professional gradient interface
- Lightning-fast performance with no external dependencies
- Polished button interactions and hover effects

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Clipboard API

1. Download or clone the repository
2. Navigate to the project folder
3. Open `index.html` in any modern browser
4. Or use Live Server in VS Code for a better experience
